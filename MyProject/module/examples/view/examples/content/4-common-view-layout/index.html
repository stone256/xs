Common Views

[ROOT]:/layout
This is the place for you website structure file, which will be shared by all pages

E.g.

  • layout
    • demo
      • header.phtml
      • footer.phtml
      • nav.phtml

  • your module template:

    <?php $page_title = 'X-FRAME | DEMO | INDEX'; ?> <?php include _X_LAYOUT . '/demo/header.phtml'; ?> <p>... </p> <?php include _X_LAYOUT . '/demo/footer.phtml'; ?>
