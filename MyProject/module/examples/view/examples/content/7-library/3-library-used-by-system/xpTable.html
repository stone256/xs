Class xpTable

  1. This class is xpPod extension mainly to deal with single table, it use same config as xpPdo
    1. $tb = new xpTable($cfg);
    2. $tb = xpTable::load('table_name', [$cfg]); //singleton

  2. Some usage compare with xpPdo except you don't need pass table name at method level. 
    
	
	$rs = xpPdo::conn()->gets('local', array('lat'>25.002102));
	
	$rs = xpTable::load('local', $cfg)->gets(array('lat'>25.002102));
