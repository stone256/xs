Class xpAS

  • This class is mainly to deal with array and string
    1. The system are heavily depends on the class to function
    2. All method can called statically by xpAS::METHOD_NAME()
    3. Some method are great to deal with mysql returned rows
    4. Note, you are not required to use those, they are just here to helpa

	 * @return node of array

	1..........0=>[
	2................id=>6

	1..........]
