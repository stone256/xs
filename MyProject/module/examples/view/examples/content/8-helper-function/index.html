Some Helpers

[ROOT]:/_system

Important system file

  • app.php - this is the main file for the system it handle routers and views
  • _system_config.php - pre-defined system constants.
    • _X_ROOT
    • _X_MODULE
    • _X_DATA
    • _X_CACHE
    • _X_LOG
    • _X_LAYOUT
    • _X_PUBLIC
    • ..
    • _X_URL
    • you can list those value by call system function:
      _defined(1); or $arr = _defined(0);
  • defalutHelper.php - some system helper functions
    • _cg($k, $ttl = 300) - cache get
    • _cp($k, $value) - cache put
    • _config('database,demo,user') - return value from global $config array
    • _controller() - return current controller
    • _defined() - return user and system defined constants
    • _dv() - print out value only if __X_DEBUG === true
    • _factory('model_name') - return model singleton instance
    • _log($data) - data to [ROOT]/_data/log/log.txt
    • _mapper() - return current url mapped route
    • _rp('/var/www/MyProject/a/b/c') - return reletive path to project root "/a/b/c"
    • _url() - return current url
  • defaultController.php - can be extand to user controllers
