Create your first project

  • Create your own module: "mymodule"
    1. rename [ROOT]/.local.sample to [ROOT]/.local and check/modify it.
    2. rename [ROOT]/config/global.php.sample to [ROOT]/config/global.php and check/modify it.
    3. create a module folder: [ROOT]/module/mymodule
    4. create router file: [ROOT]/module/mymodule/.router.php
      <?php $routers = array( '/mymodule/hello' => '/mymoudel/index@index', ); ?>
    5. create controller: [ROOT]/module/mymodule/indexController.php <?php class mymodule_indexController extends _system_defaultController{ function indexActive(){ $str = "Hello World"; return array(/** use default **/ 'data'=>array('var1'=>$str)); } } ?>
    6. create view: [ROOT]/module/mymodule/view/index/index.phtml <h1>My Modules</h1> <p><?php echo $var1; ?></p>
    7. Enable you module, add [ROOT]/config/enabled/mymodule.php
      <?php $modules[] = '/mymodule'; // this name is referred to [ROOT]/module/mymodule ?>
