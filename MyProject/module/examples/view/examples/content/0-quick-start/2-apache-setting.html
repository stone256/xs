create myfirstproject.conf under your apache config folder. <VirtualHost *:80> ServerAdmin iamadminnow@thisismysite.com DocumentRoot "/var/www/myfirstproject/public" ServerName myfirstproject.thisismysite.com #or localhost #ServerAlias localhost <Directory "/var/www/myfirstproject/public"> Options All AllowOverride All <IfVersion < 2.3 > Order allow,deny Allow from all </IfVersion> <IfVersion >= 2.3> Require all granted </IfVersion> </Directory> ErrorLog "logs/myfirstproject-error_log" CustomLog "logs/myfirstproject-access_log" common </VirtualHost>
