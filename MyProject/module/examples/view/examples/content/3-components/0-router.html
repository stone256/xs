Routers

  • File : ".router.php"

    [ROOT]/module/mymodule/.router.php

    Each module contains a key file ".router.php". this file contains all routes for current module. e.g. <?php $routers = array( /**[KEY] => [VALUE]**/ '/demo' => '/demo/demo@index', '/demo/high-light' => '/demo/demo@hilite', ); ?> /** http[s]://yoursite.com.au/demo ==>' [ROOT]/module/demo/demoController.php::indexAction() **/

  • Rule of router

    1. The key in router defination array is the absolute url path.
      e.g. "/demo"=>"xxxx" will be used as http[s]://yoursite.net.au/demo
    2. if url requested can not be found, system will try to resolve to it's parent.
      e.g. "/demo/nothing-here/there" => "xxxxxxx", if there isn't router "/demo/nothing-here/there",
      system will try to find path in follow order:
      1. /demo/nothing-here
      2. /demo
      3. /
    3. The value in router defination array indicates the controller path.
      e.g. "/demo"=>"/demo/demo@index" will resolve to [ROOT]/module/demo/democontroller.php::indexAction()
    4. when you have a router like: "/demo/abc"
      when you use /dome/abc/a=12/b=13 this will results value add to $_REQUEST Array;
      $_REQUEST = [ .. a=>12, b=>13, ] when you use /dome/abc/Australia/NSW this will results value add to $_REQUEST Array;
      $_REQUEST = [ .. Australia=>'', NSW=>'', ]
