Controllers

  1. all controller will have controller attached to it.
    e.g. demo controller will be "demoController.php"
  2. controller class name
    special character "_" (under score) used in class name
    e.g. [ROOT]/module/demo/2ndController.php has class name "demo_2ndController"
    [ROOT]/module/demo/undergroung/darkController.php has class name "demo_undergroung_darkController"
  3. controller enter point method has "Action" attached it
    e.g. "/demo/demo@index" will point to [ROOT]/module/demo/demoController.php's method indexAction() <?php class demo_demoController extends _system_defaultController { function indexAction() { ..... return array('data' => array('rs' => $rs)); } } ?>
  4. controller will only works when you put it in the .router.php
    e.g. <?php $routers = array( '/demo/high/light' => '/demo/high/light@hilite', ); ?> will activated controller : [ROOT]/module/demo/high/lightController.php, when page "/demo/high/light" requested and method :hiliteAction() in the controller will be called
