File structure

[ROOT] - your project root folder. e.g. /var/www/MyProject
  1. all call will go through /public/index.php
  2. index.php will call /.system/app.php
  3. app.php will 1st load /config/_system_config.php
  4. and /config/config.php
  5. then /.local
  6. after config loaded app will start /.system/loader.php
  7. all enabled modules info will loaded from /config/enabled/...
  8. followed by load all module's /module/mymodule/.router.php
  9. app will analyse url and routers and before calling controller /config/init.php will be loaded
  10. and call the /module/mymodule/xxxxController.php
  11. inside control you can call /module/mymodule/model/xxxx.php
  12. /module/mymodule/view/index/myxxxx.phtml will be called if the view indicated
