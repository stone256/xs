Module setup and setup script.

  1. setup and update script are set in the module's root folder.
    e.g. [ROOT]/module/demo/.setup.1.0.0.0.php
  2. all setup/update scripts are named like:
    .setup.x.x.x.x.php
    e.g. : ".setup.2.2.1.2.php"
  3. scripts ara executed in ascent order.
    e.g. script will be executed in the order :
    ".setup.1.0.0.1.php"
    ".setup.1.0.0.2.php"
    ".setup.1.0.0.3.php"
  4. after the script has been executed, ".done" will be attached to the file name to provent script running again.
    ".setup.1.0.0.3.php.done"
  5. You must give apache permission change the file name.
  6. to maintain system integrity, we strongly recommend do not output anything other than logs.
