Views

  1. all view are under view folder.
    by default, controller will use default name to find corresponding view unless specified otherwise
    e.g.
    <?php class demo_redController extends _system_defaultController { function powerAction() { ..... return array('data' => array('rs' => $rs)); } } ?> default "view" will be [ROOT]/module/demo/view/red/power.phtml
  2. specified none default view.
    e.g.
    <?php class demo_redController extends _system_defaultController { function powerAction() { ..... return array('view' => 'index.tpl.html', 'data' => array('rs' => $_SERVER)); } } ?> default "view" will be [ROOT]/module/demo/view/red/index.tpl.html
  3. all data will be passed to the view in return array's 'data' field from controller
    e.g.
    <?php class demo_redController extends _system_defaultController { function powerAction() { ..... return array('data' => array('rs' => array(a=>1,b=>2); } } ?> will show in [ROOT]/module/demo/view/red/power.phtml as variable $rs /** $rs = array(a=>1,b=>2) **/
