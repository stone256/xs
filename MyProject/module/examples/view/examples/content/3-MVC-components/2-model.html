Models

  1. all model are under folder "model", as the result it has "model" attached
    e.g.
    [ROOT]/module/demo/model/light.php will has class name "demo_model_light" <?php class demo_model_light { function get_number_of_lights_on($color='red') { ..... return $_status['on'][$color]; } } ?>
  2. model can be called use system shart cut "$myclass = _factory('demo_model_light')" as singleton
    e.g. <?php .. .. echo _factory('demo_model_light')->get_number_of_lights_on('yellow'); .. ?>
